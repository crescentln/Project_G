# Crescentln Ruleset (OpenClash + Surge)

统一维护的规则仓库，支持 OpenClash 和 Surge，自动更新。

## 基础地址

- 仓库: [https://github.com/crescentln/new-project](https://github.com/crescentln/new-project)
- Raw Base: `https://raw.githubusercontent.com/crescentln/new-project/main/ruleset/dist`

## 数据来源与更新

- 官方数据源: IANA / APNIC / Telegram CIDR / Cloudflare / AWS / GCP
- 高质量社区源: [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community)
- 工作流: `.github/workflows/ruleset-update.yml`（每 6 小时自动更新）

## 推荐格式（不是只用 txt）

- OpenClash 推荐：`openclash/<category>.yaml`、`openclash/non_ip/<category>.yaml`、`openclash/ip/<category>.yaml`
- Surge 推荐：`surge/<category>.list`、`surge/non_ip/<category>.list`、`surge/ip/<category>.list`
- `compat/*` 下的 `.txt/.conf` 主要用于兼容你原来 Sukka/Quixotic 风格链接

## 策略说明

- `DIRECT`: 直连
- `PROXY`: 走代理
- `REJECT`: 拒绝
- `REJECT-DROP`: 丢弃
- `REJECT-NO-DROP`: 拒绝（不 drop）
